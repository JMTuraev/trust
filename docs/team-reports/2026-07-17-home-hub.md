## 4. Sparkline (`sparkline.dart`)

Prototipdagi SVG uch qatlami aynan:
1. **To'ldirilgan path** ‚Äî `Path.from(line) ..lineTo(last.dx, h) ..lineTo(first.dx, h) ..close()`,
   `LinearGradient` rang **12% ‚Üí 0%** (prototip: `stop-opacity 0.12` ‚Üí `0`).
2. **Stroke** ‚Äî **`Path.cubicTo`**, Catmull-Rom ‚Üí kubik bezier aylantirish
   (`c1 = p1 + (p2-p0)/6`, `c2 = p2 - (p3-p1)/6`) ‚áí **silliq egri, siniq polyline EMAS**.
   `strokeCap/Join = round`, hero 2.2px, kichik karta 2.0px.
3. **Oxirgi nuqta** ‚Äî to'la doira (hero r=3.5, kichik karta r=3).

Rang: kirim ‚Üí `Pal.green`, chiqim ‚Üí `Pal.red`. Boshqa rang yo'q (fayl `theme.dart`
ga bog'lanmagan ‚Äî rangni chaqiruvchi beradi).

**Chekka holatlar:**
- 0 nuqta ‚Üí hech narsa chizilmaydi (`return`);
- 1 nuqta ‚Üí faqat oxirgi nuqta doirasi (chiziq yo'q);
- barcha qiymat teng (`range == 0`) ‚Üí **tekis chiziq o'rtada, NaN yo'q** (`t = 0.5`);
- `NaN`/`Infinity` kirsa `0.0` ga aylantiriladi;
- `pad = dot + stroke/2` ‚Äî prototipdagi kabi (viewBox eni 320 bo'lsa ham oxirgi
  nuqta x=310 da) doira/stroke qirqilmaydi.

`SparkDots` ‚Äî bo'sh holatdagi nuqtali ¬´kutish¬ª chizig'i (`dasharray "1 7"`, 2px,
`Pal.t6`, balandlikning 22/30 ulushida).
