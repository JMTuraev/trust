# Trust — mobil ilova (Flutter)

Prototipga (Trust.dc.html) mos, ikki tomonlama tasdiqli hisob-kitob ilovasi.
O'zbekcha, dark/light rejim, Inter shrifti.

## Lokal ishga tushirish
```bash
cd app
flutter pub get
flutter run -d chrome   # yoki -d <qurilma>
```

## Web deploy
Har `main`ga push'da GitHub Actions Flutter web'ni build qilib GitHub Pages'ga joylaydi.
Jonli URL: https://jmturaev.github.io/trust/
